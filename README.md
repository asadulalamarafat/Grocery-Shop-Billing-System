# Project title: Grocery shop billing system

# Project member: Asadul Alarm Arafat ID : 41240202286 Rizwan Oli ID : 41240202401

# Description: This C program is a Grocery Billing & Inventory Management System. It lets you add, update, search, and delete products, generate customer bills with VAT & discount, track low stock, and view total sales using a menu-driven system.

# How to run : Save the file as grocery.c. Compile it using: gcc grocery.c -o grocery. Run the program using: ./grocery (or grocery.exe on Windows).
